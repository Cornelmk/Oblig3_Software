# Oblig3_Software
Først kopierte jeg Oblig 2 og endret navn til Oblig 3. Første jeg gjorde var å legge Test filen fra Oblig 2, inn i src/test/java. Så lastet jeg Oblig 3 opp på github og trykket på Actions knappen. Der valgte jeg Java with Maven og fikk opp .github, workflows mappene og en maven.yml fil. I filen endret jeg helt nederst slik at jeg får runna testene. Jeg måtte også endre/legge til noen dependecies i pom.xml filen for at det skulle funke. Så comitta jeg og pushet det ut. Og fikk da testene opp inne på Actions-knappen.
